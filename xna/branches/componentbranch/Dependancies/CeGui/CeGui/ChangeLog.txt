CeGui# ChangeLog

~  means in progress
*  means feature is complete


== 0.5.0 RC1 ==

~ Sync the port to the upcoming CeGui 0.5.0 RC1 release
* The Direct3D renderer has been rewritten from scratch. Now uses MDX 1.0
* CeGui# is now using the Point and Size classes from the .NET Framework
* Some enums have been replaced by their System.Windows.Forms equivalents
* The Color class has been renamed to Colour to match the original CeGui
* The ColorRect class has also changes its name and is now called ColourRect
* CeGui has been broken up into several projects making it easier to manage
* Created a whole new theme specifically for CeGui#
