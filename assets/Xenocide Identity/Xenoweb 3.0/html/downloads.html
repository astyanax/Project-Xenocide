Minimum System Requirements
* Pentium IV 1800mhz or AMD Athlon 1800mhz
* 512 MB RAM Memory
* GeForce FX 5200 / ATI Readon 9200 or comparable, with latest drivers
* Windows XP or Vista
* .Net Framework 2.0 (only 3.0 is not enough)
* XNA Refresh 1.0 Runtime Distribution
* DirectX: DirectX 9.0c - be careful to not use a DirectX shipped with games, as these often lack some files

After downloading please send us feedback (positive and negative) and bug reports!

2008.04.04 - updated 2009.10.14 Progress Release 1836
Progress Release 0.4.0.1836 installer (with built-in download option for audio and high res texture files) (Size: 28.0 Mb) (http via projectxenocide.com)
This is the final installer routine. Should you still encounter any problems, please tell us about it!
The use of this installer is highly recommended, it eliminates a lot of potential problems!

If you still would rather use a zip archive, get it from here: Progress Release 0.4.0.1835 Zip archive (Size: 19.8 Mb) (http via projectxenocide.com)

Progress release 0.4.0.1835 for the first time features a playable Battleview!

The zip package comes without sound. If you wish to have Sound and Music installed as well, please download the Audio Pack (16.4 MB) (only for zip version, the installer features a download option for those files), unzip into your Xenocide Directory and enjoy!
The Xenocide Audio package seldomly changes, so there is no need to download it if you already have it!

If you decide to use the zip instead of the installer, you will need to make sure you have the follwoing software installed to run this PR:

1. DirectX 9.0c, December 2006 or newer
2. .Net Framework 2.0
3. XNA 3.0 runtime redistributable

If you have these installed, just unzip the package to any Directory of your liking and start Xenocide.exe!

Enjoy!
